# App Personal

Plataforma moderna para gestão de alunos, treinos e finanças para personal trainers.

## Funcionalidades
- Dashboard de analytics
- Onboarding multi-etapas
- Perfil do usuário completo
- Gestão financeira (pagamentos, status, histórico)
- Navegação responsiva
- Visual profissional (gradientes, glassmorphism)

## Tecnologias
- React 18 + TypeScript
- Vite
- React Router DOM
- Lucide React (ícones)
- CSS customizado

## Como rodar localmente
```bash
npm install
npm run dev
```
